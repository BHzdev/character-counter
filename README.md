# Character Counter
Este projeto consiste em um contador de caracteres simples, desenvolvido com HTML, CSS e JavaScript (usando jQuery), com o objetivo de fixar os fundamentos do jQuery.

## Funcionalidades
O Character Counter oferece as seguintes funcionalidades:

> Contagem de caracteres: o contador de caracteres exibe a quantidade de caracteres digitados em tempo real.

> Contagem de palavras: o contador de palavras exibe a quantidade de palavras digitadas em tempo real.

## Como usar
Clone o repositório em sua máquina local usando o seguinte comando:

`git clone https://github.com/bhzdev/Character-Counter.git`

Abra o arquivo index.html em seu navegador.

Comece a digitar no campo de texto fornecido e observe a contagem de caracteres e palavras atualizando em tempo real.

Se desejar, você pode definir um limite de caracteres para o campo de texto, digitando um número no campo "Limite de caracteres". O contador de caracteres exibirá quantos caracteres ainda podem ser digitados antes de atingir o limite.
